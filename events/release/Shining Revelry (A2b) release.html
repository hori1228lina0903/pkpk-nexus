Shining Revelry (A2b) release

Release date
Mar 27, 2025 - Ongoing

Shining Revelry (A2b) release is now available! This expansion introduces new ex cards and exclusive rewards.

New cards from this set can be obtained by opening Shining Revelry (A2b) booster packs. Complete missions to earn Emblem Tickets and special items!

Game Updates

  • New solo step-up Battles (see below)
  • Cards from Triumphant Light (A2a) become available for trading.
  • The number of decks in the builder has been increased from 15 to 20.
  • Minor qualify of life updates (bigger font on some cards, improvements on effects...)
