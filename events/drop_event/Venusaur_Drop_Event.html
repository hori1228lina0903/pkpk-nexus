Venusaur Drop Event

Event period
Nov 29 - Dec 13, 2024

The Venusaur Drop Event is a limited-time solo battle event. During the event period, you can challenge special battles to obtain promo cards and other rewards. Complete all tasks in each battle to earn bonus rewards!

Promo Pack

Pack name
Promo Pack A Series Vol. 2

These promo cards can be obtained as rewards from event battles.
