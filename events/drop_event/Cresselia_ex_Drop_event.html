Cresselia ex Drop event

Event period
Feb 3 - Feb 17, 2025

The Cresselia ex Drop event is a limited-time solo battle event. During the event period, you can challenge special battles to obtain promo cards and other rewards. Complete all tasks in each battle to earn bonus rewards!

Event Battles

Feb 3 - Feb 17, 2025
