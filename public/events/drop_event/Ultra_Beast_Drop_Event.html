Ultra Beast Drop Event

Event period
Jun 4 - Jun 14, 2025

The Ultra Beast Drop Event is a limited-time solo battle event. During the event period, you can challenge special battles to obtain promo cards and other rewards. Complete all tasks in each battle to earn bonus rewards!

Promo Pack

Pack name
Promo Pack A Series Vol. 9

These promo cards can be obtained as rewards from event battles.
